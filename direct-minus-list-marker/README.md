# Пометка фраз по списку минус-слов

Находит точные совпадения фраз из выгрузки Директа со списком минус-фраз и пишет `удалить` в колонку `Параметр 1`.

## Запуск

```bash
python mark_phrases_by_minus_list.py
```

## Входные файлы
- `direct.csv — выгрузка Директа`
- `minus_words.txt — фразы для пометки`

## Выходные файлы
- `output.csv — выгрузка с пометками`

В репозитории оставлены только демонстрационные данные. Перед рабочим запуском замените примерные файлы своими выгрузками.

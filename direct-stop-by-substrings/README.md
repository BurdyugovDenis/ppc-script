# Остановка фраз по подстрокам

Ищет заданные подстроки в колонке с фразами выгрузки Директа и проставляет `остановить` в `Параметр 1`.

## Запуск

```bash
python mark_stop_by_substrings.py
```

## Входные файлы
- `direct_export_input.csv — выгрузка Директа`
- `stop_contains.txt — список подстрок`

## Выходные файлы
- `direct_export_output.csv — файл с пометками`

## Зависимости
- `pandas`

В репозитории оставлены только демонстрационные данные. Перед рабочим запуском замените примерные файлы своими выгрузками.

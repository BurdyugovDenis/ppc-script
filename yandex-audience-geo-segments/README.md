# Создание геосегментов Яндекс Аудиторий

Создает сегменты типа геолокация-окружности через API Яндекс Аудиторий, разбивая координаты на пакеты.

## Запуск

```bash
YANDEX_AUDIENCE_TOKEN=your_token python create_geo_segments.py
```

## Входные файлы
- `locations.txt — координаты `широта,долгота``

## Выходные файлы
- `Сегменты создаются в Яндекс Аудиториях; локального output-файла нет`

## Зависимости
- `requests`

## Источник
- https://vc.ru/marketing/2150647-sozdanie-geotochok-v-yandeks-auditoriyakh-s-pomoshchyu-skripta-api

## Важно
- Запускайте сначала на маленьком наборе координат.

В репозитории оставлены только демонстрационные данные. Перед рабочим запуском замените примерные файлы своими выгрузками.

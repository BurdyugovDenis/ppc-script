# Поиск терминов из списка 2 в списке 1

Сравнивает запросы и термины через лемматизацию русских и английских слов. Подходит для поиска пересечений между большим списком запросов и словарем терминов.

## Запуск

```bash
python find_terms_in_queries.py
```

## Входные файлы
- `list1.csv — колонка query`
- `list2.csv — колонка term`

## Выходные файлы
- `result_matches.csv — флаги и детали совпадений`

## Зависимости
- `pandas`
- `pymorphy3`
- `nltk`

В репозитории оставлены только демонстрационные данные. Перед рабочим запуском замените примерные файлы своими выгрузками.

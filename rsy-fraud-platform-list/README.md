# Список фродовых площадок РСЯ

Фильтрует площадки РСЯ по маскам вроде `game`, `vpn`, `dsp`, сортирует по расходу и исключает разрешенные домены.

## Запуск

```bash
python build_rsy_fraud_platforms.py
```

## Входные файлы
- `input.csv — колонки Площадка;Расход (руб.)`

## Выходные файлы
- `output.csv — площадки и расход без заголовка`

## Зависимости
- `pandas`

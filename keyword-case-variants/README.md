# Генератор падежных вариантов ключей

Генерирует падежные и числовые формы фраз, а также варианты с `!`, `[]` и `[!]` для Яндекс Директа.

## Запуск

```bash
python generate_keyword_case_variants.py
```

## Входные файлы
- `input.txt — ключи по одному на строку`

## Выходные файлы
- `output.txt — варианты ключей`

## Зависимости
- `pymorphy3`
- `pymorphy3-dicts-ru`

В репозитории оставлены только демонстрационные данные. Перед рабочим запуском замените примерные файлы своими выгрузками.

# Автоматическое минусование площадок РСЯ через Google Sheets

README-шаблон по публичной Google Sheet из статьи. Публичный экспорт таблицы содержит структуру листов, но не отдает привязанный Apps Script-код, поэтому сам Code.gs нужно копировать из связанного шаблона Google Sheets.

## Запуск

```bash
Скопируйте Google Sheet из статьи, откройте Extensions -> Apps Script, проверьте настройки на листе settings и первый запуск делайте с test_mode=true.
```

## Входные файлы
- `settings — пример в settings_example.csv`
- `Базовые список площадок (топ 300 по расходу)`
- `Placements_All`
- `Exclusions_Log`

## Выходные файлы
- `Обновленные исключения площадок в кампаниях`
- `Exclusions_Log с причинами`

## Важно
- Полный текст статьи не копируется дословно в репозиторий; оставлены ссылка, краткая инструкция и пример структуры.
- Bound Apps Script недоступен через публичный export Google Sheets без доступа к проекту Apps Script.

В репозитории оставлены только демонстрационные данные. Перед рабочим запуском замените примерные файлы своими выгрузками.

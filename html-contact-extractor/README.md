# Извлечение телефонов и email из HTML

Читает HTML-таблицу, нормализует телефоны в формат `7XXXXXXXXXX`, собирает уникальные телефоны и email.

## Запуск

```bash
python extract_contacts_from_html.py
```

## Входные файлы
- `contacts.html — HTML-таблица с контактами`

## Выходные файлы
- `contacts_clean.csv`
- `phones.txt`
- `emails.txt`

## Зависимости
- `beautifulsoup4`
- `pandas`

В репозитории оставлены только демонстрационные данные. Перед рабочим запуском замените примерные файлы своими выгрузками.

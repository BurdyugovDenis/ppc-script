# Фильтр фраз по минус-словам

Фильтрует ключевые фразы по обычным минус-словам, фразам в кавычках и фразам в квадратных скобках с учетом лемматизации.

## Запуск

```bash
python filter_by_minus_words.py
```

## Входные файлы
- `input.txt — фразы по одной на строку`
- `minus.txt — минус-слова и минус-фразы`

## Выходные файлы
- `result.txt — фразы, попавшие под минуса`
- `white.txt — чистые фразы`

## Зависимости
- `pymorphy3`

В репозитории оставлены только демонстрационные данные. Перед рабочим запуском замените примерные файлы своими выгрузками.

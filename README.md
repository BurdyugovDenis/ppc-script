# Скрипты для Яндекс Директа, РСЯ, SEO и Google Sheets

Это очищенная GitHub-версия набора рабочих скриптов. Исходные папки переименованы по функционалу, реальные выгрузки заменены короткими демонстрационными файлами, секреты вынесены из кода, а у каждого скрипта есть свой README.

## Как пользоваться

1. Откройте нужную папку.
2. Прочитайте локальный `README.md`.
3. Установите зависимости при необходимости:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

4. Замените примерные `input.*`/`settings_example.csv` своими рабочими данными.

## Каталог

Полный список лежит в `SCRIPT_INVENTORY.csv`. В нем указаны новые папки, назначение, исходная локальная папка и статья-источник, если она была.

## Обновление скриптов

Если рабочий скрипт изменился, обновляйте соответствующий файл в папке с понятным названием и повторно проверяйте:

```bash
python tools/check_no_large_examples.py
rg -n 'y0__|YA_DIRECT_TOKEN_REAL|ANTICAPTCHA_KEY = "[0-9a-f]' --glob '!README.md' .
```

Перед публикацией не добавляйте полные выгрузки, реальные телефоны, e-mail, токены, client login клиентов и HTML-экспорты с контактами. В репозитории должны оставаться только короткие примеры.

## Источники

Часть скриптов связана со статьями vc.ru. Тексты статей не копируются дословно в README: в папках оставлены ссылки, краткие инструкции и очищенные/примерные файлы.

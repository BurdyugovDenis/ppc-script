# Проверка выдачи по доменам конкурентов

Selenium-скрипт открывает выдачу Яндекса по фразам и проверяет, встречаются ли в HTML выдачи домены конкурентов.

## Запуск

```bash
python check_competitor_domains.py
```

## Входные файлы
- `phrases.txt — запросы по одному на строку`
- `domain.txt — домены конкурентов`

## Выходные файлы
- `results.txt — статус целевое/не целевое`
- `progress.txt — прогресс обработки`

## Зависимости
- `selenium`

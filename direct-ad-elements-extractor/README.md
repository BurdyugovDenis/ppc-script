# Выгрузка уникальных элементов объявлений

Извлекает уникальные ссылки, заголовки, тексты, быстрые ссылки, описания и уточнения из CSV-выгрузки Директ Коммандера.

## Запуск

```bash
python extract_unique_ad_elements.py
```

## Входные файлы
- `input.csv — UTF-16 CSV из Директ Коммандера`

## Выходные файлы
- `output.txt — `значение;название столбца``

# GUI-анализатор уникальных фраз

Tkinter-приложение сравнивает два списка ключевых фраз. Фразы нормализуются через pymorphy3, стоп-слова отбрасываются, порядок слов не влияет на сравнение.

## Запуск

```bash
python phrase_analyzer.py
```

## Входные файлы
- `input1.txt — пример существующих фраз`
- `input2.txt — пример новых фраз`

## Выходные файлы
- `output.txt — уникальные новые фразы`

## Зависимости
- `pymorphy3`

# Сбор ключей по кампаниям

Удаляет дубликаты пар `campaign` + `keyword` и собирает все ключи каждой кампании в одну строку.

## Запуск

```bash
python collect_keywords_by_campaign.py
```

## Входные файлы
- `input.csv — столбцы campaign;keyword`

## Выходные файлы
- `output.csv — кампания и объединенный список ключей`

## Зависимости
- `pandas`

В репозитории оставлены только демонстрационные данные. Перед рабочим запуском замените примерные файлы своими выгрузками.

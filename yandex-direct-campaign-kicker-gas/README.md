# Пиналка кампаний Яндекс Директа

Google Apps Script проверяет клики за вчера по заданным кампаниям. Если кликов не больше лимита, скрипт делает suspend + resume и пишет лог в Google Sheets.

## Запуск

```bash
Откройте Google Sheets -> Apps Script, вставьте campaign_kicker.gs и повесьте hourly trigger на kickTick().
```

## Входные файлы
- `Лист settings — пример в settings_example.csv`

## Выходные файлы
- `Лист kick_log — лог запусков`

## Источник
- Локальная папка `Пиналка`

## Важно
- OAuth-токен храните в Google Sheets/Script Properties, не публикуйте его в коде.

В репозитории оставлены только демонстрационные данные. Перед рабочим запуском замените примерные файлы своими выгрузками.

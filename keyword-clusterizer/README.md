# Кластеризатор ключевых фраз

Лемматизирует фразы, удаляет предлоги, считает вес слов и присваивает группе название по наиболее значимым словам.

## Запуск

```bash
python cluster_keywords.py
```

## Входные файлы
- `input.csv — `фраза;частотность`, без заголовка`
- `excluded_words.csv — исключаемые слова`

## Выходные файлы
- `output.csv — сгруппированные фразы`
- `value_keyword.csv — веса слов`

## Зависимости
- `pymorphy3`

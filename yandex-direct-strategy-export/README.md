# Выгрузка стратегий кампаний

Два варианта скрипта выгружают стратегии текстовых кампаний Яндекс Директа. Расширенная версия дополнительно подтягивает регионы групп объявлений.

## Запуск

```bash
YANDEX_DIRECT_TOKEN=token YANDEX_DIRECT_CLIENT_LOGIN=login python export_strategies_with_regions.py
```

## Входные файлы
- `Переменные окружения YANDEX_DIRECT_TOKEN и YANDEX_DIRECT_CLIENT_LOGIN`

## Выходные файлы
- `direct_strategies.csv — пример структуры результата`

## Зависимости
- `requests`
- `pandas`

## Важно
- Реальные OAuth-токены удалены из кода.

В репозитории оставлены только демонстрационные данные. Перед рабочим запуском замените примерные файлы своими выгрузками.

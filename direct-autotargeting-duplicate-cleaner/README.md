# Чистка дублей автотаргетинга

Удаляет строки `---autotargeting` в группах, где автотаргетинг встретился два раза и более, затем нормализует номер группы.

## Запуск

```bash
python clean_duplicate_autotargeting.py
```

## Входные файлы
- `input.csv — выгрузка Директа с шапкой на третьей строке`

## Выходные файлы
- `output.csv — очищенная выгрузка`

## Зависимости
- `pandas`

В репозитории оставлены только демонстрационные данные. Перед рабочим запуском замените примерные файлы своими выгрузками.

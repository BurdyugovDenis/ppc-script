# Частотный словарь слов

Разбивает список фраз на отдельные слова и считает количество вхождений каждого слова.

## Запуск

```bash
python build_word_frequency.py
```

## Входные файлы
- `phrases.csv — фразы в первом столбце`

## Выходные файлы
- `word_counts.csv — слово и число упоминений`
